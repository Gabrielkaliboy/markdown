# vscode一些其他设置

## 设置代码对齐线
用VS Code 写代码的时候有时候缩进太多就不知道对应的是哪一个标签了，可不可以让它显示缩进参考线，方法如下：

找到 文件-->首选项——>设置→搜索renderIntentGuides→将此选项改为true(默认为false)，就可以了。
